# Internship Tracker

A static internship hour tracker for GitHub Pages.

## Defaults

- Goal: 320 hours
- Start date: June 15, 2026
- No-duty dates: June 22 and June 23, 2026
- Duty schedule: Monday to Friday
- Forecast: 8 hours per future duty day

## Use

Open `index.html` in a browser. Your entries are saved in localStorage for that browser.

## Deploy To GitHub Pages

1. Put these files in a GitHub repository.
2. In GitHub, open Settings -> Pages.
3. Choose your branch and root folder.
4. Save, then open the published Pages URL.

## Backup

Use the download button in the app to export a JSON backup. Use the upload button to restore it.
